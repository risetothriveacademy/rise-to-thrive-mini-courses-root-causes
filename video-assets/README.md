# Rise to Thrive – Video Assets

This folder standardizes all video + social assets for the 18 Mini-Courses
WITHOUT duplicating any teaching scripts.

- Teaching scripts: live in the main repo (e.g. `mini-courses/ads/...`)
- Video assets: live here under `video-assets/`

## Structure

- `mini-course-index.md` – map of course numbers → script files
- `docs/`
  - `fliki-master-template.md` – universal scene structure for Fliki
  - `social-caption-kit.md` – universal caption templates
  - `va-instructions.md` – step-by-step for VAs
- `templates/`
  - `fliki-scenes.template.md` – per-course Fliki scene template
  - `youtube-description.template.md` – per-course YT description template
  - `social-captions.template.md` – per-course caption stub
  - `ad-script.template.md` – per-course ad stub (no script duplication)
- `courses/##_*`
  - one folder per mini-course (01–18)
  - VAs copy templates into each folder and fill placeholders
