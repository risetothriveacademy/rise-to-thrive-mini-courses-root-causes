## YouTube SEO Best Practices

### 1. **Title Optimization**
- Front-load primary keywords
- Keep under 60 characters
- Use emotional hooks + practical promise
- Include year for evergreen content (optional)
- Format: "Emotional Hook | Practical Benefit + Keyword"

### 2. **Description Optimization**
- First 150 characters = preview (most critical)
- Include primary keyword in first sentence
- Add 3-5 timestamps for long-form videos
- Include 2-3 CTAs (course link, subscribe, Foundation)
- Add relevant hashtags (3-5 max)
- Include Scripture references for SEO + credibility

### 3. **Tags Strategy**
- 15-20 tags per video
- Mix of:
  - Broad keywords (e.g., "mental health")
  - Specific long-tail (e.g., "christian grief recovery")
  - Branded tags (e.g., "rise to thrive")
  - Competitor tags (research similar channels)

### 4. **Thumbnail Design**
- Face + emotion = connection
- Text overlay = 3-5 words max
- High contrast colors (avoid red/green if face present)
- Consistent branding across series
- Test variations for best CTR

### 5. **Playlist Organization**
- Create themed playlists by pain point
- Add to "Rise to Thrive - Mini Courses" master playlist
- Use playlist descriptions for additional keywords

## Performance Tracking

Track these metrics weekly:
- CTR (Click-Through Rate): Goal 5-10%
- AVD (Average View Duration): Goal 50%+
- Watch Time: Higher = better ranking
- Conversion Rate: Video views → Course purchases
