## Playlist SEO Tips

### For Each Playlist:
1. **Title**: Include primary keyword (e.g., "Christian Anxiety Relief")
2. **Description**:
   - First line = keyword-rich summary
   - Include CTA to Foundation Course
   - Add relevant hashtags
3. **Custom Thumbnail**: Use consistent branding
4. **Settings**:
   - Set as "Public"
   - Add to channel homepage
   - Enable "Add new videos automatically" if using keyword matching
